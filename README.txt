Car Rental Management System - TDS4223 Project

How to run:
1. Open this folder in VS Code or Dev C++.
2. Compile Project_GR1.cpp with g++.
3. Keep all .txt files in the same folder as the executable.
4. In VS Code, press F5 to build and run Project_GR1_run.exe.

Login accounts:
Staff/Admin: admin / admin123
Staff/Admin: staff / staff123
Customer: customer / cust123

The system uses cars.txt, staff_users.txt, customer_users.txt, rentals.txt,
payments.txt, reports.txt, maintenance.txt, receipts.txt, feedback.txt,
activity_log.txt, promotions.txt, and incidents.txt.
